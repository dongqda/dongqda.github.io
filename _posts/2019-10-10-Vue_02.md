# II. Sample Vue

###### 코드

```html
<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8">
    <title>vue sample</title>
  </head>
  <body>
    <div id="app">
      {{message}}
    </div>
    <script src="https://cdn.jsdelivr.net/npm/vue@2.5.2/dist/vue.js"></script>

  <script>
    new Vue({
      el:'#app',
      data:{
        message:'Hello Vue.js'
      }
    });

  </script>
  </body>
</html>
```

